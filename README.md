
# 🧠 Teachable Machine Model Project
## 📌 About

 This project hosts a Machine Learning model created using Google Teachable Machine. The model is trained on custom examples and runs directly in the browser using TensorFlow.js.

## 🚀 Live Demo

 👉 Live Website: [https://teachablemachine.withgoogle.com/models/bLQqw0ehv/](https://teachablemachine.withgoogle.com/models/bLQqw0ehv/)

## ⚙️ Tech Stack

- Teachable Machine (Model Training)

- TensorFlow.js (Model Execution in Browser)

- HTML5

- CSS3

- JavaScript

## 📂 Model Files

- model.json – Model architecture (used by TensorFlow.js)

- metadata.json – Model labels and metadata

- model.weights.bin – Trained model weights

## 🛠 How It Works

The model is trained on examples (such as images, sounds, or poses). Its predictions depend on the quality and quantity of the training data provided.
